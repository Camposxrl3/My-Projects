# Software Development IV
This project was developed as part of the course requirements and focuses on learning and applying web development concepts using JSP and frontend technologies.

## Project Overview
The main goal of this project was to design and implement a functional web application using HTML, JSP, and CSS, while understanding basic client–server interaction and database connectivity.

The application establishes a connection with a MySQL database, allowing dynamic data handling and reinforcing core web architecture concepts.

## The project demonstrates the ability to:
- Develop dynamic web pages using JSP
- Create structured user interfaces with HTML and CSS
- Handle user input and server-side processing
- Establish and manage MySQL database connections
- Apply fundamental software development principles

## Technologies Used
- JSP (Java Server Pages)
- Java
- HTML / CSS
- MySQL
- Intellij IDEA (IDE)

## Notes
This project was developed for academic purposes and represents my learning progress during the course. While it may not follow all production-level best practices, it demonstrates a solid foundation in web development, JSP usage, and database integration.
