# Software Development IX
This project was developed as part of the course requirements and focuses on building a modern full-stack web application using JavaScript technologies, RESTful architecture, and third-party service integration.

## Project Overview
The main goal of this project was to design and implement a functional web application using JavaScript, applying modern backend development concepts such as REST API development, database modeling, payment processing, and containerization.

A REST API was developed using Fastify to manage communication between the frontend and the database. MongoDB was used as the database system, with Mongoose handling schema definition and data modeling. Docker was implemented to run MongoDB inside a container, providing a consistent development environment.

Additionally, Stripe was integrated to handle online payments, allowing secure transaction processing within the application.

## The project demonstrates the ability to:
- Develop frontend interfaces using HTML, CSS, and JavaScript
- Build RESTful APIs using Fastify
- Handle client–server communication
- Integrate third-party services such as Stripe for payments
- Model and manage data using MongoDB
- Use environment variables for secure configuration
- Implement containerized services using Docker
- Apply modern software development and web architecture principles

## Technologies Used
- JavaScript
- HTML
- CSS
- Fastify (REST API)
- MongoDB
- Mongoose
- Stripe (Payment Processing)
- Docker

## Required Modules
The following modules are required to run the project using pnpm:

pnpm add fastify
pnpm add @fastify/cors
pnpm add mongoose
pnpm add stripe
pnpm add dotenv
pnpm add @fastify/static
